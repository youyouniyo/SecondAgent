AI Readability Checker

Confluence 문서를 AI가 읽기 좋은 문서로 점검하세요

Confluence page URL을 입력하면 문서 구조와 표현을 분석합니다. 예상 소요 시간: 약 30초-2분

AI-readable 문서는 무엇이 다를까요?

사람이 읽기 쉬운 문서와 AI가 안정적으로 이해하는 문서는 조금 다릅니다. AI-readable 문서는 절차, 조건, 예외, 용어가 명확해서 검색, 요약, 자동화, Agent 실행에 더 잘 활용됩니다.

명확한 구조

제목, 표, 목록의 역할이 분명해야 합니다

문서의 제목 계층, 절차 목록, 표 데이터가 구분되어 있으면 AI가 문서의 흐름과 근거를 안정적으로 파악할 수 있습니다.
